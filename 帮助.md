# django-restframework-vue
django rest framework + vue + redis 抽奖业务实现API小项目。前后端分离


# 环境
- Python 3.8
 - django 4.0.1
 - restframework 3.12.4
 - redis 4.0.0
 - drf_yasg 1.20.0 
 - PyJwt 1.7.1
 
 # 功能 
 - 注册-登录
 
 - 增加抽奖名单-抽奖-抽奖奖品-等级业务实现
 
 - 查看抽奖记录-删除抽奖
